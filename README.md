# Helicopter-trajectory-tracking-with-ADRC-control

## Introduction

This project presents a MATLAB-based visualization environment for analysing the trajectory tracking performance of a helicopter model inspired by the Quanser 2-DOF Helicopter system. The framework combines geometric motion visualization with angular response plots to provide an intuitive representation of the system behaviour.

The objective is to study how the helicopter responds to commanded pitch and yaw inputs and to visually assess the tracking quality of the implemented controller.

---

## Methodology

The helicopter motion is represented in two dimensions using the pitch and yaw responses generated by the control system. At each simulation step:

- Yaw determines the orientation of the helicopter in the horizontal plane
- Pitch influences the evolution of the trajectory
- The instantaneous vehicle position is updated and rendered in real time

A geometric helicopter model is superimposed on the path to provide a clear representation of the vehicle orientation and motion.

---

### Trajectory Tracking

The primary panel displays the commanded and measured trajectories in the X-Y plane. The desired path is represented independently from the actual response, allowing immediate visual assessment of tracking performance.

The helicopter is modelled using the following graphical elements:

| Component | Function |
|-----------|-----------|
| Main body | Indicates the central structure of the vehicle |
| Tail boom | Represents the rear extension used for orientation reference |
| Main rotor | Indicates the lifting rotor location |
| Nose indicator | Shows the instantaneous heading direction |

The difference between the reference trajectory and the measured trajectory provides a direct indication of controller accuracy.

---

### Pitch Response

The upper-right panel shows the temporal evolution of the pitch angle. By comparing the commanded and measured pitch signals, the following characteristics can be evaluated:

- Rise time
- Overshoot
- Steady-state error
- Response lag

---

### Yaw Response

The lower-right panel presents the yaw response over time. This plot provides insight into the directional behaviour of the helicopter and highlights the quality of heading control.

---

## MATLAB Implementation

The simulation framework is designed to support the evaluation of reference tracking performance in a multi-variable helicopter system. The implementation combines trajectory generation, signal comparison, and geometric state representation within a unified environment.

The MATLAB implementation includes:

-Generation of periodic reference commands for both pitch and yaw channels
-Computation and visualization of the corresponding system response
-Simultaneous comparison between commanded and measured states
-Two-dimensional mapping of helicopter motion from pitch-yaw dynamics
-Time-synchronized plotting of trajectory, pitch, and yaw responses
-Graphics handle-based updating for computationally efficient real-time execution
